About This Page

This page is styled using only CSS to demonstrate the Box Model. Notice the use of margins, borders, and padding on different elements.

Margin
Border
Padding
Content

Content Area

This is where your actual content lives. Dimensions are controlled by width and height properties.

Padding

Creates space inside the element, between the content and border. Affects background color.

Border

Surrounds the padding and content. Can be styled with different colors, widths, and styles.

Margin

Creates transparent space outside the border, separating the element from its neighbors.
